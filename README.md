计算机安全的定义
       1）保密性
          1）数据保密性：确保隐私或者秘密信息不向非授权者泄漏，也不被非授权者所使用
          2）隐私性
             确保个人能够控制或确定与自身相关的哪些信息是可以被收集，被保存的，这些信息可以由谁来公开以及向谁公开
       2）完整性
          数据完整性：确保信息和程序只能以特定和授权的方式进行改变
          系统完整性：确保系统以一种正常的方式来执行预定的功能，免于有意或者无意的非授权操纵
       3）可用性
          确保系统能工作迅速，对授权用户不能拒绝服务。
